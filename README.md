# Scala with scalatest.
## Usage
### Master branch
```
$ sbt new nabezokodaikon/scala-basic.g8
```
or
```
$ g8 nabezokodaikon/scala-basic.g8
```


### Other branch
```
$ g8 nabezokodaikon/scala-basic -b feature
```


## Test
```
$ g8 file://scala-basic.g8/ --name=scala-basic-test --force
$ cd scala-basic-test
```
