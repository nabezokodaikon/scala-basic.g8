# Scala with scalatest.
## Usage
### Master branch
```bash
$ g8 nabezokodaikon/scala-sbt
```
### Other branch
```bash
$ g8 nabezokodaikon/scala-sbt -b feature
```
## Test
```bash
$ g8 file://scala-sbt.g8/ --name=uftest --force
$ cd uftest
```
